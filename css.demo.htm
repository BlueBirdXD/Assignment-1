CSS Properties Demonstrated

  • Color and background-color for styling text and backgrounds
  • Padding and margin for spacing
  • Border and border-radius for borders and rounded corners
  • Flexbox for flexible layouts
  • CSS Grid for grid-based layouts
  • Box-shadow for depth and dimension
  • Font properties like font-family and color

Simple Example

HTML:
<div class="card">
  <h3>Title</h3>
  <p>Content here</p>
</div>

CSS:
.card { padding: 20px; border: 2px solid #ddd; border-radius: 5px; }
